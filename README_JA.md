# Jisui - kindle2pdf

Kindle for PCで表示されている書籍を自動スキャンし、Remarkable2などの電子ペーパー端末に最適化されたPDFを生成するツールです。

## 🚀 使い方

![UI Image](https://github.com/RyuheiRay/Jisui-kindle2pdf/blob/master/src/assets/screenshot.png)

1. **ダウンロードと準備**
   - [Releases](https://github.com/RyuheiRay/Jisui-kindle2pdf/releases) から最新の `Jisui-kindle2pdf.zip` をダウンロードして展開（解凍）します。

2. **アプリを起動する**
   - 展開したフォルダ内の `Jisui-kindle2pdf.exe` をダブルクリックして起動します。
   - **警告が出る場合**: 「WindowsによってPCが保護されました」と表示されることがありますが、これは自作ソフトに対する標準的な警告です。**「詳細情報」**をクリックし、**「実行」**を選択してください。

3. **Kindleの準備（左のボタン）**
   - アプリ左側の **「1.Open Kindle」** ボタンを押してKindleを起動します。
   - 変換したい本を開いてください。

4. **【重要】表示設定の手動調整**
   - **文字の調整**: 閲覧デバイスで読みやすくなるよう、Kindleアプリの「Aa」メニューから**文字サイズ、フォント、幅、余白を必ず手動で調整**してください。
   - **理由**: 本ツールは「画面に表示されているそのまま」を保存するため、この調整が仕上がりの読みやすさに直結します。
   - **表示モード**: 必ず「1ページ表示」に設定してください。

5. **変換スタート（真ん中のボタン）**
   - 真ん中の **矢印ボタン（2.Start!）** を押すと変換が始まります。
   - **【注意】** 実行時に自動で先頭ページへ移動し、全画面化されます。キャプチャが終わるまでは、マウスやキーボードに触れずに放置してください。

6. **PDFを確認する（右のボタン）**
   - 完了後、右側の **「3.Open Folder」** ボタンを押すと `result` フォルダが開かれ、PDFを確認できます。

## 💡 安定して実行するためのヒント
- **手動クリック**: Kindleが自動で手前に来ない場合は、Startを押した直後にKindleの画面を一度クリックしてください。
- **スリープ禁止**: PCが途中でスリープしないよう設定を確認してください。
