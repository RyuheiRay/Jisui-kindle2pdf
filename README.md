# Jisui - kindle2pdf

Kindle for PCで開いている書籍を自動スキャンし、Remarkable2などの電子ペーパー端末に最適なPDFを作成するツールです。

## 🚀 使い方



1. **アプリを起動する**
   - ダウンロードしたフォルダ内の `Jisui-kindle2pdf.exe` をダブルクリックして起動します。

2. **Kindleの準備（左のボタン）**
   - アプリ左側の **「1.Open Kindle」** ボタンを押してKindleを起動します。
   - 変換したい本を開きます。

3. **【重要】表示設定の手動調整**
   - **文字の大きさや幅の調整**: 使用するデバイス（Remarkable2等）で読みやすくなるよう、Kindleアプリ上の「Aa」メニューから**手動でフォントサイズや行間、余白などを調整**してください。このツールは「画面に表示されているそのまま」をキャプチャするため、ここでの調整が仕上がりに直結します。
   - **表示モード**: 1ページ表示（見開きではない状態）に設定してください。

4. **変換スタート（真ん中のボタン）**
   - 真ん中の **矢印ボタン（2.Start!）** を押すと変換が始まります。
   - **【注意】** キャプチャが終わるまでは、マウスやキーボードに触れずにそのまま放置してください。

5. **PDFを確認する（右のボタン）**
   - キャプチャが終わったら、右側の **「3.Open Folder」** ボタンを押します。
   - 開かれたフォルダの中に、完成したPDFが保管されています。

---

## 💡 安定して実行するためのヒント
- **放置が成功のコツ**: スキャン中にマウスを動かしてしまうと、画像がズレたりエラーの原因になります。
- **スリープ設定**: ページ数が多い本を変換する場合、処理中にPCがスリープしないようご注意ください。
